First Java Project “HelloWorld”

  1. After launching Eclipse, close the welcome tab (if it appears)
  2. On the left-hand side you should see a panel with a tab called “Package Explorer”.
  3. Right click on any empty space within this panel
  4. Move your mouse over “New” Then click “Java Project” (this tutorial doesn’t cover Maven Repositories. For maven you have to make a very similar “Maven Project” which will be discussed later)
  5. Name your project anything. First project is typically named “HelloWorld” (without quotes).
  6. Click “Finish” in the bottom right-hand corner
  7. Double-click the newly created project in the “Package Explorer” panel
  8. Right-click the “src” folder
  9. Move your mouse over “New“. Then click “Class” (You can name the class the same as the project name)
  10. Uncheck the checkbox asking to create a module. You probably don’t need it. I’ve never needed it.
  11. Click Finish
  12. Standard first test project (what I’m doing now) is to just print out “Hello World!” into the console. Use the code below to do that
public class HelloWorld {
//Main method, executes your code, just memorize it for now
    public static void main(String[]args) {
        //Prints out the text in quotes
        System.out.println("Hello World!");
    }
}
