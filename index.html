Generic code for various research projects

  1. CSVStripper.java Quickly strips lines from a CSV matching your criteria and prints those lines to a separate csv. How this program works:
    1. First the program will ask you to select the CSV file to be searched
    2. Then it will read the csv header to determine what fields are present
    3. Then it will display a GUI with n rows, where n is equal to the number of fields detected. Each row will contain the first value for a field, and then an empty inputbox
    4. Text you type into the inputbox is what the program will look for in the specified field. If it finds the text you asked for, it will print the whole line (and all other lines that match as well) to a separate CSV.
  2. CSVStripperPerformance.java Same as CSVStripper but for a folder of CSV files

Large Language Models (LLM): This might sound complicated but it's actually really easy to set up. There's no code you need to write because other, much smarter people did all the hard work for you.

  • Pre-requisites:
    • Money (jk everything here is free online, but there are minimum hardware requirements here for high-end use of LLMs)
    • You technically don't need a graphics processing unit (GPU/graphics card), but you'll have slower responses, and you won't be able to run higher-end LLMs
    • The systems I have done this with are:
      DesktopLaptop
      CPUi7-12700i7-10750H
      RAM32GB DDR5 5200MHz16GB DDR4 3200MHz
      GPURTX 3060 Ti1660 Ti (Mobile)
      VRAM8GB DDR66GB DDR6

  • If you have an NVIDIA GPU Download from this link
